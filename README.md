# PUiRCHR_Tests

**Платформа для самопроверки знаний и умений в области психологии управления и развития человеческих ресурсов.**  

Программа представляет собой комплекс тестов, позволяющих оценить ключевые навыки эффективного руководителя: способность к делегированию, мотивации персонала, адаптации сотрудников и другим аспектам управления людьми.

## Системные требования

| Компонент | Требования |
|-----------|------------|
| **Операционная система** | Windows 11 |
| **Разрешение экрана** | 1920?1080 (рекомендуется для 15.6" дисплеев) |
| **Дополнительное ПО** | Microsoft Visual Studio 2022 |

### **Установка необходимых компонентов:**

#### 1. Microsoft Visual Studio 2022
**Скачайте и установите:**
- Перейдите на [официальный сайт Visual Studio](https://visualstudio.microsoft.com/vs/community/)
- Нажмите **"Скачать Community 2022"**
- Запустите установщик

#### 2. Выбор рабочих нагрузок
**При установке обязательно выберите:**
-  Разработка приложений для ПК с помощью .NET
-  ASP.NET и веб-разработка
-  Кроссплатформенная разработка .NET

## Установка и запуск программы

1. Скачайте архив "PUiRCHR_Tests.zip";
2. Создайте новую папку на Вашем ПК;
3. Откройте архив и извлеките все файлы в созданную папку;
4. Перейдите в папку с распакованными файлами.
5. Запустите файл "Quizes2.sln" в программе Microsoft Visual Studio 2022.
